¿Cómo funciona la interfaz?

El proceso de conexión es simple y seguro, permitiendo el envío de operaciones de SAE a COI sin requerimientos tecnológicos adicionales.

  • Conexión Segura: El contador invita al usuario de SAE para establecer un enlace con un ID único.
  • Contabilización Dinámica: Se generan pólizas contables de manera automática con base en la información recibida.
  • Visibilidad Completa: El contador tiene acceso a todos los movimientos para un control total.

Novedades en Detalle

Conexión Directa SAE v10 - COI v11

La nueva interfaz permite un flujo de trabajo simplificado. Desde COI, el contador puede invitar al usuario de SAE para establecer una conexión segura entre ambas empresas.

Esta conexión se realiza de forma remota, sin la necesidad de configuraciones complejas o requerimientos tecnológicos adicionales.

Contabilización Automática de Pólizas

Una vez conectado, el sistema permite la generación automática de pólizas contables con base en las operaciones de SAE. El contador configura las cuentas y las pólizas dinámicas desde COI, manteniendo el control total.

Esto se puede configurar para que suceda de forma automática, permitiendo que la información fluya sin intervención manual.

Operación en Sucursales Centralizada

La nueva versión de Siigo Aspel COI 11 simplifica la organización de las operaciones en sucursales, permitiendo una gestión más eficiente y consolidada de la contabilidad.

Centraliza el control y la visibilidad de todas las operaciones, sin importar su origen.
